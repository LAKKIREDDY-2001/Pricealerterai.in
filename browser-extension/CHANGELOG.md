# Changelog

All notable changes to the AI Price Tracker browser extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial release of AI Price Tracker browser extension
- One-click price tracking from supported shopping sites
- Instant browser notifications when prices drop
- Price history and trends visualization
- Auto sync with web dashboard
- Dark mode support
- Affiliate link integration for Amazon, Flipkart, and Myntra
- Context menu integration for quick actions
- Support for 8 shopping platforms:
  - Amazon (India & International)
  - Flipkart
  - Myntra
  - Ajio
  - Meesho
  - Snapdeal
  - Tata CliQ
  - eBay

### Features
- Popup interface with tracker management
- Real-time price extraction from product pages
- Target price setting with quick percentage buttons (10%, 15%, 20%, 25%)
- User authentication state management
- Local storage for offline functionality
- Server sync when online
- Sound alerts option
- Push notification controls

### Technical
- Manifest V3 compliant
- Service worker for background tasks
- Content scripts for price extraction
- Alarms for periodic price checking (hourly)
- Context menus for quick actions
- Toast notifications for user feedback

## [1.0.1] - 2024-02-01

### Improved
- Enhanced price extraction algorithms
- Better error handling for network failures
- Updated affiliate link handling

### Fixed
- Dark mode styling issues
- Notification permission request flow

## [1.1.0] - 2024-03-01
