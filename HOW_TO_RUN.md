# How to Run

1.  **Activate virtual environment (if exists):**
    ```bash
    source venv/bin/activate
    ```

2.  **Install dependencies:**
    ```bash
    pip install Flask Flask-Cors requests beautifulsoup4 Werkzeug
    ```
3.  **Run the Flask application:**
    ```bash
    python app.py
    ```
4.  **Access the application:**
    Open your browser and navigate to `http://127.0.0.1:8081`. You will be redirected to the signup page. After signing up, you can log in and access the dashboard.
